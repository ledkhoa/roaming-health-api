## Description

POC for Roaming Health -- Connecting healthcare workers with open shifts.

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# e2e tests
$ pnpm run exp:e2e
```
